# DevCamper API

> Backend API for DevCamper application, which is a bootcamp directory website.

## Usage

Rename "config/confog.env.env" to "config/config.env" and update the values/settings to your own.

## Install dependencies
```
npm install
```

## Run app
```
# Run in dev mode
npm run dev

# Run is prod mode
mpn start
```
